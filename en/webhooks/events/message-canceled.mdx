---
title: "message.canceled"
description: "Queued or scheduled message canceled before completion."
---

`message.canceled` is emitted when Rewrite successfully cancels a queued or scheduled message.

```json
{
  "id": "748395130237498916",
  "createdAt": "2026-03-19T18:42:30.000Z",
  "type": "message.canceled",
  "data": {
    "id": "748395130237498505",
    "projectId": "748395130237498412",
    "to": "+5511999999999",
    "tags": [{ "name": "campaign", "value": "billing_reminder" }],
    "type": "SMS",
    "status": "CANCELED",
    "content": "Rewrite: your invoice is due tomorrow",
    "country": "br",
    "analysis": {
      "characters": 36,
      "encoding": "gsm7",
      "segments": { "concat": 153, "count": 1, "reason": "fits", "single": 160 }
    },
    "error": null,
    "deliveredAt": null,
    "scheduledAt": "2026-03-19T20:00:00.000Z",
    "templateId": null
  }
}
```

Notes:

- This event is emitted by the public cancel flow.
- `data.scheduledAt` may remain populated when the canceled message was originally scheduled.
