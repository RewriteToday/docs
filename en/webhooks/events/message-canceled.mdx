---
title: "message.canceled"
description: "Scheduled message canceled before completion."
---

`message.canceled` is emitted when Rewrite successfully cancels a cancelable future-delivery message.

```json
{
  "id": "748395130237498916",
  "createdAt": "2026-03-19T18:42:30.000Z",
  "type": "message.canceled",
  "sandbox": false,
  "data": {
    "id": "748395130237498505",
    "projectId": "748395130237498412",
    "contact": "Fernanda",
    "contactId": "748395130237498515",
    "to": "+5511999999999",
    "tags": { "campaign": "billing_reminder" },
    "sandbox": false,
    "type": "SMS",
    "status": "CANCELED",
    "content": "Rewrite: your invoice is due tomorrow",
    "country": "br",
    "analysis": {
      "characters": 37,
      "encoding": "gsm7",
      "segments": { "concat": 153, "count": 1, "reason": "fits", "single": 160 }
    },
    "error": null,
    "deliveredAt": null,
    "scheduledAt": "2026-03-19T20:00:00.000Z",
    "templateId": null
  }
}
```

Notes:

- This event is emitted by the public cancel flow.
- Public cancelation is meant for scheduled messages.
- `data.scheduledAt` can remain populated after cancelation.
