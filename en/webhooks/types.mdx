## SMS Events

<div style={{ display: 'grid', gap: '26px' }}>

  <div>
    <div>— <a href="/webhooks/events/sms-queued"><code style={{ color: '#B5B3AD' }}>sms.queued</code></a></div>
    <div style={{ marginLeft: '18px', marginTop: '6px', opacity: 0.85 }}>
      Occurs when a **SMS** is accepted and placed into the delivery queue.
    </div>
  </div>

  <div>
    <div>— <a href="/webhooks/events/sms-delivered"><code style={{ color: '#3DD68C' }}>sms.delivered</code></a></div>
    <div style={{ marginLeft: '18px', marginTop: '6px', opacity: 0.85 }}>
      Occurs when the carrier confirms successful delivery to the recipient device.
    </div>
  </div>

  <div>
    <div>— <a href="/webhooks/events/sms-scheduled"><code style={{ color: '#FFCA16' }}>sms.scheduled</code></a></div>
    <div style={{ marginLeft: '18px', marginTop: '6px', opacity: 0.85 }}>
      Occurs when a **SMS** is scheduled for delivery at a future time.
    </div>
  </div>

  <div>
    <div>— <a href="/webhooks/events/sms-failed"><code style={{ color: '#FF9592' }}>sms.failed</code></a></div>
    <div style={{ marginLeft: '18px', marginTop: '6px', opacity: 0.85 }}>
      Occurs when the **SMS** failed permanently due to invalid numbers, blocked routes, quota limits, or provider rejection.
    </div>
  </div>

  <div>
    <div>— <a href="/webhooks/events/sms-canceled"><code style={{ color: '#BAA7FF' }}>sms.canceled</code></a></div>
    <div style={{ marginLeft: '18px', marginTop: '6px', opacity: 0.85 }}>
      Occurs when a scheduled **SMS** was canceled before it was sent.
    </div>
  </div>

</div>
