Rewrite Theme for Neovim is the official Rewrite theme for Neovim.

Repository: [github.com/RewriteToday/theme](https://github.com/RewriteToday/theme)

## When to use it

- You want the Rewrite palette in Neovim.
- You prefer installing editor themes through your plugin manager from GitHub.
- You want an automatic variant that follows `vim.o.background`.
- You want matching `lualine` support.

## Install

Use your plugin manager to install the repository root:

```lua
{
  "RewriteToday/theme",
  name = "rewrite.nvim",
  lazy = false,
  priority = 1000,
  config = function()
    require("rewrite").setup({ style = "auto" })
    vim.cmd.colorscheme("rewrite")
  end,
}
```

## Activate a variant

Use the automatic mode:

```lua
vim.cmd.colorscheme("rewrite")
```

Force dark or light explicitly:

```lua
vim.cmd.colorscheme("rewrite-night")
vim.cmd.colorscheme("rewrite-day")
```

Enable the matching `lualine` theme:

```lua
require("lualine").setup({
  options = {
    theme = "rewrite",
  },
})
```
