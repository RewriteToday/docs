Rewrite Theme for Visual Studio Code is the official Rewrite theme extension for VS Code.

Repository: [github.com/RewriteToday/theme](https://github.com/RewriteToday/theme)

## When to use it

- You use Visual Studio Code or Cursor-compatible VS Code theming flows.
- You want a direct marketplace install.
- You want a focused dark variant and a warmer light variant with the same accent language.
- You want a known extension id for managed editor setups.

## Install

Open the Extensions view inside Visual Studio Code, search for `Rewrite Theme`, and install it directly from the Marketplace.

```text
Extensions -> Search: Rewrite Theme -> Install
```

If you prefer searching by identifier, use:

```text
rewritetoday.rewrite
```

## Activate a variant

Open `Preferences: Color Theme` and choose:

```text
Rewrite Night
Rewrite Day
```

## Package contents

```text
vscode/
├─ package.json
├─ themes/rewrite-night-color-theme.json
├─ themes/rewrite-day-color-theme.json
└─ scripts/package.sh
```
