Rewrite Theme for Zed is the official Rewrite theme extension for Zed.

Repository: [github.com/RewriteToday/theme](https://github.com/RewriteToday/theme)

## When to use it

- You use Zed as your primary editor.
- You want a direct marketplace install with no manual file setup.
- You want both Rewrite theme variants in one extension package.
- You want to switch between dark and light mode from the Zed theme selector.

## Install

1. Open Zed.
2. Open the Extensions view.
3. Search for `Rewrite Theme`.
4. Click `Install`.

## Activate a variant

After installing, open the theme selector in Zed and choose:

- `Rewrite Night`
- `Rewrite Day`

## Package contents

```text
zed/
├─ extension.toml
└─ themes/rewrite.json
```
