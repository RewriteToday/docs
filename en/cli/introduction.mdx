The **Rewrite CLI** is the official command-line tool to interact with Rewrite directly from your terminal.

Built for developers, it lets you send SMS, manage resend flows, listen to webhooks, simulate deliveries, and automate messaging pipelines — all without leaving your shell.

## Installation

<CodeGroup>
```bash Go (recommended)
go install github.com/rewritetoday/cli@latest
```
```bash Brew
brew install rewrite-cli
```
```bash Docker
docker pull rewrite/cli
```
</CodeGroup>

## Verify installation

```bash
rewrite --version
```

<Steps>
  <Step title="Authenticate">
      ```bash
      rewrite login
      ```
      Authenticate using your **API key** or **OAuth flow**.
  </Step>
  <Step title="Send a SMS">
      ```bash
      rewrite messages send \
      "Hello from Rewrite" \
      --to +5511999999999 \
      ```
  </Step>
  <Step title="Listen for webhooks">
    ```bash
    rewrite listen --forward-to http://localhost:3000/webhooks/rewrite
    ```
    Events will stream in real time and optionally forward to your local server.
  </Step>
</Steps>

## Design Principles
Rewrite CLI was built to be:

- Explicit — No hidden behavior
- Scriptable — Predictable structured output
- CI-friendly — Fully non-interactive via flags
- Fast — Minimal startup time and low footprint
