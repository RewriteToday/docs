## Before you start

- You need to have Node.js installed on your machine.
- You need to have a Rewrite API key with the permissions.


<Steps>
    <Step title="Install the SDK" titleSize="h3">
        To install the Node.js SDK, run the following command:
        
        <CodeGroup>
        
        ```bash npm
        npm install @rewritejs/sdk
        ```
        
        ```bash yarn
        yarn add @rewritejs/sdk
        ```
        
        ```bash pnpm
        pnpm add @rewritejs/sdk
        ```
        
        ```bash bun
        bun install @rewritejs/sdk
        ```
        
        </CodeGroup>
    </Step>
    
    <Step title="Set up REWRITE_API_KEY environment variable" titleSize="h3">
    Set up your Rewrite API key in your environment variables.
    
    ```text .env    
    REWRITE_API_KEY=rw...
    ```
    </Step>
    
    <Step title="Send your SMS" titleSize="h3">
    Now, all you need to do is pass the message content and that's it!
    
    ```ts sms.ts
      import { Rewrite } from '@rewritejs/sdk';

      const rewrite = new Rewrite(process.env.REWRITE_API_KEY);

      async function sendSms(to: string) {
        const params = {
          to,
          message: 'Your OTP code is 478201',
        };

        const { data, error } = await rewrite.messages.send(params);

        if (error) {
          throw error;
        }

        return data;
      }

      async function main() {
        const message = await sendSms('+1234567890');
        console.log(message);
      }

      main().catch(console.error);
    ```
    
    You can do everything with our SDK, keep exploring!
    </Step>
</Steps>
