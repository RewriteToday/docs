Rewrite Theme for Visual Studio Code e a extensao oficial de tema da Rewrite para VS Code.

Repositorio: [github.com/RewriteToday/theme](https://github.com/RewriteToday/theme)

## Quando usar

- Quando voce usa Visual Studio Code ou fluxos compativeis com temas do VS Code.
- Quando quer instalacao direta via marketplace.
- Quando quer uma variante dark focada e uma variante light mais quente, com a mesma linguagem visual.
- Quando precisa de um extension id conhecido para setups gerenciados.

## Instalacao

Abra a view de Extensions dentro do Visual Studio Code, procure por `Rewrite Theme` e instale direto do Marketplace.

```text
Extensions -> Search: Rewrite Theme -> Install
```

Se preferir buscar pelo identificador, use:

```text
rewritetoday.rewrite
```

## Ativar uma variante

Abra `Preferences: Color Theme` e escolha:

```text
Rewrite Night
Rewrite Day
```

## Conteudo do pacote

```text
vscode/
├─ package.json
├─ themes/rewrite-night-color-theme.json
├─ themes/rewrite-day-color-theme.json
└─ scripts/package.sh
```
