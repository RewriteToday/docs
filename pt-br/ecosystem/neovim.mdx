Rewrite Theme for Neovim e o tema oficial da Rewrite para Neovim.

Repositorio: [github.com/RewriteToday/theme](https://github.com/RewriteToday/theme)

## Quando usar

- Quando voce quer a paleta da Rewrite dentro do Neovim.
- Quando prefere instalar temas via plugin manager apontando para o GitHub.
- Quando quer uma variante automatica que segue `vim.o.background`.
- Quando quer suporte combinado para `lualine`.

## Instalacao

Use seu plugin manager para instalar a raiz do repositorio:

```lua
{
  "RewriteToday/theme",
  name = "rewrite.nvim",
  lazy = false,
  priority = 1000,
  config = function()
    require("rewrite").setup({ style = "auto" })
    vim.cmd.colorscheme("rewrite")
  end,
}
```

## Ativar uma variante

Use o modo automatico:

```lua
vim.cmd.colorscheme("rewrite")
```

Force dark ou light explicitamente:

```lua
vim.cmd.colorscheme("rewrite-night")
vim.cmd.colorscheme("rewrite-day")
```

Ative o tema correspondente no `lualine`:

```lua
require("lualine").setup({
  options = {
    theme = "rewrite",
  },
})
```
