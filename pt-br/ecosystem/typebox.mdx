[`@rewritetoday/typebox`](https://www.npmjs.com/package/@rewritetoday/typebox) fornece schemas oficiais em TypeBox para a API da Rewrite, permitindo validar requests, responses, recursos reutilizaveis e payloads de webhook em runtime.

Repositorio: [github.com/RewriteToday/typebox](https://github.com/RewriteToday/typebox)

## Quando usar

- Quando voce prefere TypeBox como camada de validacao em runtime.
- Quando quer validacao em runtime alem dos tipos TypeScript.
- Quando recebe input externo antes de chamar endpoints da Rewrite.
- Quando precisa de schemas versionados alinhados aos modelos da API Rewrite.

## Exemplo de validacao de request

```ts
import { RESTPostCreateTemplateBody } from '@rewritetoday/typebox/v1';
import { Parse } from '@sinclair/typebox/value';

const data = Parse(RESTPostCreateTemplateBody, {
  name: 'order_shipped',
  content: 'Hi {{first_name}}, your order {{order_id}} is on the way.',
  variables: [
    { name: 'first_name', fallback: 'customer' },
    { name: 'order_id', fallback: '0000' },
  ],
});

console.log(data);
```

## Exemplo de validacao de webhook

```ts
import { WebhookEvent } from '@rewritetoday/typebox/v1';
import { Parse } from '@sinclair/typebox/value';

let event;

try {
  event = Parse(WebhookEvent, payload);
} catch (error) {
  console.error({ error });
  throw new Error('Payload de webhook Rewrite invalido');
}

console.log({ data: event });
```

## Boas praticas

<Accordion title="Mantenha versoes explicitas">
  Importe de um namespace especifico, como `@rewritetoday/typebox/v1`, para manter upgrades intencionais.
</Accordion>

<Accordion title="Valide nas bordas">
  Faca parse do input externo antes de ele chegar na logica de negocio para falhar cedo e com previsibilidade.
</Accordion>

<Accordion title="Combine validacao e transporte">
  Use `@rewritetoday/typebox` junto com `@rewritetoday/rest` e `@rewritetoday/types` para fechar validacao, rotas e transporte na mesma stack.
</Accordion>
