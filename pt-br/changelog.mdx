# 2026-05-06

- Adicionada a secao dedicada [Sandbox](/pt-br/sandbox/introduction) com visao geral e [magic numbers](/pt-br/sandbox/magic-numbers).
- Adicionada a pagina dedicada de [Inbound](/pt-br/webhooks/inbound) dentro de Webhooks.
- Removida a cobertura publica de grupos nao publicos como contatos, segmentos e API keys.
- Exemplos atualizados para usar metadata em objeto em vez de arrays de pares.

# 2026-04-21

- Documentados os filtros reais de `GET /messages`: `status`, `country`, `encoding`, `scheduled` e `withCounts`.
- Adicionado `RECEIVED` na cobertura do ciclo de vida de mensagens.

# 2026-04-20

- Adicionada a cobertura do webhook [`message.received`](/pt-br/webhooks/events/message-received).
- Trafego inbound passou a ser documentado como contrato publico de evento, sem expor callbacks de provider.

# 2026-04-03

- Recursos de webhook atualizados para usar `isEnabled` no lugar de `status`.
- Adicionada a cobertura de delete em lote com `DELETE /webhooks`.
- Adicionados `POST /templates/{id}/duplicate` e `DELETE /templates` na referencia publica.

# 2026-04-02

- Referencias legadas de `/webhooks/{id}/logs` foram substituidas por `/webhooks/{id}/deliveries`.
- Adicionada cobertura publica para `GET /deliveries` e `GET /deliveries/{id}`.
- Logs de request e entregas de webhook passaram a aparecer separados na referencia publica.
