title: Introducao
description: Aprenda a usar a API da Rewrite.
---

A API da **Rewrite** permite enviar e gerenciar SMS de forma simples e confiavel.

Use os endpoints da secao **SMS** para:

- enviar mensagens
- listar mensagens
- consultar uma mensagem por ID
- reenviar ou cancelar mensagens

Consulte tambem a pagina de [paginacao](/pt-br/api/pagination) para navegar resultados com cursor.
