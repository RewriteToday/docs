# Visao geral

Os endpoints de listagem da Rewrite usam **paginacao por cursor**.

Toda resposta paginada retorna:

- `data`: os itens da pagina atual
- `cursor`: o estado de paginacao da proxima ou da pagina anterior

## Formato da resposta

```json
{
  "ok": true,
  "data": [],
  "cursor": {
    "persist": true,
    "next": "748395130237498399",
    "prev": "748395130237498420"
  }
}
```

## Campos do cursor

`persist`: Indica se existem mais resultados.<br/>
`next`: Cursor da proxima requisicao.<br/>
`prev`: Cursor da requisicao anterior.

- Trate `cursor.next` e `cursor.prev` como valores opacos.

## Parametros de query

- `after`: Busca resultados depois de um ID.
- `before`: Busca resultados antes de um ID.
- `limit`: Quantidade de itens retornados (2 a 100).

## Paginacao para frente

Use `after` para avancar.

<CodeGroup>
    ```bash CLI
    rewrite webhooks list --limit 25 --after 9213923891293040
    ```

    ```ts Node.js
    const rewrite = new Rewrite(process.env.REWRITE_API_KEY);

    const { data, error, cursor } = await rewrite.webhooks.list({
      limit: 25,
    });

    if (error) throw error;

    if (cursor?.persist) {
      const { data: nextData, error: nextError, cursor: nextCursor } =
        await rewrite.webhooks.list({
          limit: 25,
          after: cursor.next,
        });

      if (nextError) throw nextError;
    }
    ```

    ```go Golang
    package main

    import (
        rewrite "github.com/rewritetoday/golang"
    )

    func main() {
        client := rewrite.New("rw_abcxyz")

        data, cursor, err := client.Webhooks.List(25)
        if err != nil {
            panic(err)
        }

        if cursor != nil && cursor.Persist {
            nextData, nextCursor, err := client.Webhooks.List(
                25,
                rewrite.Webhooks.ListOptions{After: cursor.Next},
            )
            if err != nil {
                panic(err)
            }

            _, _ = nextData, nextCursor
        }

        _ = data
    }
    ```

    ```py Python
    import rewrite

    client = rewrite.Rewrite("rw_abcxyz")

    data, error, cursor = client.webhooks.list(limit=25)

    if error:
        raise Exception(error)

    if cursor and cursor.persist:
        next_data, next_error, next_cursor = client.webhooks.list(
            limit=25,
            after=cursor.next,
        )

        if next_error:
            raise Exception(next_error)
    ```

    ```bash cURL
    curl -X GET 'https://api.rewritetoday.com/v1/webhooks?limit=25&after=9213923891293040' \
         -H 'Authorization: Bearer rw_abcxyz'
    ```

    ```bash HTTPie
    http GET 'https://api.rewritetoday.com/v1/webhooks?limit=25&after=9213923891293040' \
         'Authorization: Bearer rw_abcxyz'
    ```
</CodeGroup>

## Erros comuns

<Tabs sync={false}>
  <Tab title="Usar before e after juntos">
      Nao envie `before` e `after` na mesma requisicao.
  </Tab>
  <Tab title="Inferir o fim pela quantidade">
      Nao use `length < limit` para detectar fim de lista. Use `cursor.persist`.
  </Tab>
  <Tab title="Gerar cursor no client">
      Nao gere cursor no client. Reutilize os valores retornados pela API.
  </Tab>
</Tabs>
