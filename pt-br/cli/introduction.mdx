A **Rewrite CLI** e a ferramenta oficial de linha de comando para interagir com a Rewrite diretamente do seu terminal.

Feita para desenvolvedores, ela permite enviar SMS, gerenciar reenvios, ouvir webhooks, simular entregas e automatizar pipelines de mensageria sem sair do shell.

## Instalacao

<CodeGroup>
```bash Go (recomendado)
go install github.com/rewritetoday/cli@latest
```
```bash Brew
brew install rewrite-cli
```
```bash Docker
docker pull rewrite/cli
```
</CodeGroup>

## Verificar instalacao

```bash
rewrite --version
```

<Steps>
  <Step title="Autenticar">
      ```bash
      rewrite login
      ```
      Autentique com sua **API key** ou via **OAuth flow**.
  </Step>
  <Step title="Escutar webhooks">
    ```bash
    rewrite listen --forward-to http://localhost:3000/webhooks/rewrite
    ```
    Os eventos serao exibidos em tempo real e podem ser encaminhados para seu servidor local.
  </Step>

  <Step title="Enviar um SMS">
      ```bash
      rewrite trigger sms.created -m "Hello from Rewrite" --to +5511999999999
      ```
  </Step>
</Steps>

## Principios de design

A Rewrite CLI foi criada para ser:

- Explicita - Sem comportamento escondido
- Scriptavel - Saida estruturada e previsivel
- CI-friendly - Totalmente nao interativa via flags
- Rapida - Inicializacao minima e baixo consumo
