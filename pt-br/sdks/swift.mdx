## Antes de comecar

- Voce precisa de Swift 5.9+ instalado na sua maquina.
- Voce precisa de uma API key da Rewrite com as permissoes necessarias.

<Steps>
    <Step title="Instalar o SDK" titleSize="h3">
        Adicione a dependencia do pacote Swift:

        ```swift Package.swift
        dependencies: [
          .package(url: "https://github.com/rewritetoday/swift-sdk", from: "1.0.0")
        ]
        ```
    </Step>

    <Step title="Configurar a variavel REWRITE_API_KEY" titleSize="h3">
    Configure sua API key da Rewrite nas variaveis de ambiente.

    ```text .env
    REWRITE_API_KEY=rw...
    ```
    </Step>

    <Step title="Enviar seu SMS" titleSize="h3">
    Agora e so passar o conteudo da mensagem.

    ```swift main.swift
    import Foundation
    import RewriteSDK

    @main
    struct Main {
      static func main() async throws {
        let rewrite = Rewrite(apiKey: ProcessInfo.processInfo.environment["REWRITE_API_KEY"] ?? "")

        let message = try await rewrite.messages.send(
          to: "+1234567890",
          message: "Your OTP code is 478201"
        )

        print(message)
      }
    }
    ```

    Voce pode fazer tudo com nosso SDK. Continue explorando.
    </Step>
</Steps>
