## Antes de comecar

- Voce precisa ter Node.js instalado na sua maquina.
- Voce precisa de uma API key da Rewrite com as permissoes necessarias.

<Steps>
    <Step title="Instalar o SDK" titleSize="h3">
        Para instalar o SDK de Node.js, rode o comando abaixo:
        
        <CodeGroup>
        
        ```bash npm
        npm install @rewritetoday/sdk
        ```
        
        ```bash yarn
        yarn add @rewritetoday/sdk
        ```
        
        ```bash pnpm
        pnpm add @rewritetoday/sdk
        ```
        
        ```bash bun
        bun install @rewritetoday/sdk
        ```
        
        </CodeGroup>
    </Step>
    
    <Step title="Configurar a variavel REWRITE_API_KEY" titleSize="h3">
    Configure sua API key da Rewrite nas variaveis de ambiente.
    
    ```text .env    
    REWRITE_API_KEY=rw...
    ```
    </Step>
    
    <Step title="Enviar seu SMS" titleSize="h3">
    Agora e so passar o conteudo da mensagem.
    
    ```ts sms.ts
      import { Rewrite } from '@rewritetoday/sdk';

      const rewrite = new Rewrite(process.env.REWRITE_API_KEY);

      async function sendSms(to: string) {
        const params = {
          to,
          content: 'Your OTP code is 478201',
        };

        const { data, error } = await rewrite.messages.send(params);

        if (error) {
          throw error;
        }

        return data;
      }

      async function main() {
        const message = await sendSms('+1234567890');
        console.log(message);
      }

      main().catch(console.error);
    ```
    
    Voce pode fazer tudo com nosso SDK. Continue explorando.
    </Step>
</Steps>
