## Eventos de SMS

<div style={{ display: 'grid', gap: '26px' }}>

  <div>
    <div>— <a href="/pt-br/webhooks/sms/queued"><code style={{ color: '#B5B3AD' }}>sms.queued</code></a></div>
    <div style={{ marginLeft: '18px', marginTop: '6px', opacity: 0.85 }}>
      Ocorre quando um **SMS** e aceito e colocado na fila de entrega.
    </div>
  </div>

  <div>
    <div>— <a href="/pt-br/webhooks/sms/delivered"><code style={{ color: '#3DD68C' }}>sms.delivered</code></a></div>
    <div style={{ marginLeft: '18px', marginTop: '6px', opacity: 0.85 }}>
      Ocorre quando a operadora confirma a entrega no dispositivo do destinatario.
    </div>
  </div>

  <div>
    <div>— <a href="/pt-br/webhooks/sms/scheduled"><code style={{ color: '#FFCA16' }}>sms.scheduled</code></a></div>
    <div style={{ marginLeft: '18px', marginTop: '6px', opacity: 0.85 }}>
      Ocorre quando um **SMS** e agendado para envio futuro.
    </div>
  </div>

  <div>
    <div>— <a href="/pt-br/webhooks/sms/failed"><code style={{ color: '#FF9592' }}>sms.failed</code></a></div>
    <div style={{ marginLeft: '18px', marginTop: '6px', opacity: 0.85 }}>
      Ocorre quando o **SMS** falha permanentemente por numero invalido, rota bloqueada, limite de cota ou rejeicao do provedor.
    </div>
  </div>

  <div>
    <div>— <a href="/pt-br/webhooks/sms/canceled"><code style={{ color: '#BAA7FF' }}>sms.canceled</code></a></div>
    <div style={{ marginLeft: '18px', marginTop: '6px', opacity: 0.85 }}>
      Ocorre quando um **SMS** agendado e cancelado antes do envio.
    </div>
  </div>

</div>
