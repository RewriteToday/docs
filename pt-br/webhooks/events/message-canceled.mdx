---
title: "message.canceled"
description: "Mensagem agendada cancelada antes da conclusao."
---

`message.canceled` e emitido quando a Rewrite cancela com sucesso uma mensagem cancelavel de entrega futura.

```json
{
  "id": "748395130237498916",
  "createdAt": "2026-03-19T18:42:30.000Z",
  "type": "message.canceled",
  "sandbox": false,
  "data": {
    "id": "748395130237498505",
    "projectId": "748395130237498412",
    "contact": "Fernanda",
    "contactId": "748395130237498515",
    "to": "+5511999999999",
    "tags": { "campaign": "billing_reminder" },
    "sandbox": false,
    "type": "SMS",
    "status": "CANCELED",
    "content": "Rewrite: sua fatura vence amanha",
    "country": "br",
    "analysis": {
      "characters": 32,
      "encoding": "gsm7",
      "segments": { "concat": 153, "count": 1, "reason": "fits", "single": 160 }
    },
    "error": null,
    "deliveredAt": null,
    "scheduledAt": "2026-03-19T20:00:00.000Z",
    "templateId": null
  }
}
```

Notas:

- Esse evento vem do fluxo publico de cancelamento.
- O cancelamento publico e voltado para mensagens agendadas.
- `data.scheduledAt` pode continuar preenchido apos o cancelamento.
