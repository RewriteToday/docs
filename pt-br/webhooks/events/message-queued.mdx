---
title: "message.queued"
description: "Mensagem imediata persistida antes do dispatch."
---

`message.queued` e emitido assim que a Rewrite persiste um envio imediato para entrega.

```json
{
  "id": "748395130237498913",
  "createdAt": "2026-03-19T18:42:11.000Z",
  "type": "message.queued",
  "sandbox": false,
  "data": {
    "id": "748395130237498500",
    "projectId": "748395130237498412",
    "contact": "Fernanda",
    "contactId": "748395130237498515",
    "to": "+5511999999999",
    "tags": { "flow": "otp_login", "origin": "checkout" },
    "sandbox": false,
    "type": "SMS",
    "status": "QUEUED",
    "content": "Rewrite: seu codigo de verificacao e 123456",
    "country": "br",
    "analysis": {
      "characters": 43,
      "encoding": "gsm7",
      "segments": { "concat": 153, "count": 1, "reason": "fits", "single": 160 }
    },
    "error": null,
    "deliveredAt": null,
    "scheduledAt": null,
    "templateId": null
  }
}
```

Notas:

- `message.queued` e usado apenas para envios imediatos.
- Envios agendados usam [`message.scheduled`](./message-scheduled).
- Projetos em sandbox tambem podem emitir esse evento quando um magic number força o resultado `QUEUED`.
