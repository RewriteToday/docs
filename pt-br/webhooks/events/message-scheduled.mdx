---
title: "message.scheduled"
description: "Mensagem aceita com horario futuro em `scheduledAt`."
---

`message.scheduled` e emitido quando a Rewrite aceita uma mensagem com `scheduledAt` no futuro.

```json
{
  "id": "748395130237498914",
  "createdAt": "2026-03-19T18:42:11.000Z",
  "type": "message.scheduled",
  "data": {
    "id": "748395130237498503",
    "projectId": "748395130237498412",
    "to": "+5511999999999",
    "tags": [{ "name": "campaign", "value": "billing_reminder" }],
    "type": "SMS",
    "status": "SCHEDULED",
    "content": "Rewrite: sua fatura vence amanha",
    "country": "br",
    "analysis": {
      "characters": 33,
      "encoding": "gsm7",
      "segments": { "concat": 153, "count": 1, "reason": "fits", "single": 160 }
    },
    "error": null,
    "deliveredAt": null,
    "scheduledAt": "2026-03-19T20:00:00.000Z",
    "templateId": null
  }
}
```

Notas:

- `data.scheduledAt` vem preenchido.
- Eventos posteriores podem ser [`message.sent`](./message-sent), [`message.failed`](./message-failed) ou [`message.canceled`](./message-canceled).
