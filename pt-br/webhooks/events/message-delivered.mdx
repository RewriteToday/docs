---
title: "message.delivered"
description: "Confirmacao final de entrega pelo provider."
---

`message.delivered` e emitido quando a Rewrite recebe confirmacao final de entrega do provider.

```json
{
  "id": "748395130237498916",
  "createdAt": "2026-03-19T18:42:39.000Z",
  "sandbox": false,
  "type": "message.delivered",
  "data": {
    "id": "748395130237498500",
    "projectId": "748395130237498412",
    "sandbox": false,
    "contact": "Fernanda",
    "contactId": "748395130237498515",
    "to": "+5511999999999",
    "tags": { "use_case": "otp_login" },
    "type": "SMS",
    "status": "DELIVERED",
    "content": "Rewrite: seu codigo de verificacao e 123456",
    "country": "br",
    "analysis": {
      "characters": 43,
      "encoding": "gsm7",
      "segments": { "concat": 153, "count": 1, "reason": "fits", "single": 160 }
    },
    "error": null,
    "deliveredAt": "2026-03-19T18:42:39.914Z",
    "scheduledAt": null,
    "templateId": null
  }
}
```

Notas:

- `data.status` e `DELIVERED`.
- `data.deliveredAt` vem preenchido.
- Em sandbox, magic numbers podem forcar esse estado sem trafego real.
