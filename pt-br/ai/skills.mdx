Skills da Rewrite sao pacotes reutilizaveis de capacidade para agentes de AI.
Eles definem fluxos focados, instrucoes reaproveitaveis e padroes de execucao para o time entregar mais rapido com qualidade previsivel.

## Como instalar e usar Skills

Use o comando abaixo para adicionar uma fonte de skill:

```bash
npx skills add <link>
```

Exemplo com os Skills da Rewrite:

```bash
npx skills add https://github.com/RewriteToday/skills
```

<Info>
Depois de adicionar, siga a documentacao do seu ambiente de agente para listar, ativar e executar as skills instaladas.
</Info>

## Estrutura tipica de uma skill

<Accordion title="1. Escopo e objetivo">
  Defina exatamente o que a skill cobre e quando ela deve ser acionada.
</Accordion>

<Accordion title="2. Fluxo de execucao">
  Descreva a sequencia de acoes: descoberta, implementacao, validacao e handoff.
</Accordion>

<Accordion title="3. Guardrails">
  Inclua regras obrigatorias como checagens de seguranca, estilo e operacoes proibidas.
</Accordion>

<Accordion title="4. Ativos reutilizaveis">
  Adicione scripts, templates e referencias para reduzir repeticao manual.
</Accordion>

## Boas praticas

- Mantenha cada skill focada em um unico resultado.
- Prefira passos concretos em vez de instrucoes genericas.
- Defina criterios de validacao objetivos.
- Versione e evolua skills junto com seu codigo e suas ferramentas.
- Conecte cada skill a fluxos reais do seu time.
