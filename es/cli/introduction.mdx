La **CLI de Rewrite** es la herramienta oficial de línea de comandos para interactuar con Rewrite directamente desde tu terminal.

Creada para desarrolladores, te permite enviar SMS, gestionar flujos de reenvío, escuchar webhooks, simular entregas y automatizar pipelines de mensajería, todo sin salir de tu shell.

## Instalación

<CodeGroup>
```bash Go (recommended)
go install github.com/rewritetoday/cli@latest
```
```bash Brew
brew install rewrite-cli
```
```bash Docker
docker pull rewrite/cli
```
</CodeGroup>

## Verifica la instalación

```bash
rewrite --version
```

<Steps>
  <Step title="Autentícate">
      ```bash
      rewrite login
      ```
      Autentícate usando tu **API key** o el **flujo OAuth**.
  </Step>
  <Step title="Escucha webhooks">
    ```bash
    rewrite listen --forward-to http://localhost:3000/webhooks/rewrite
    ```
    Los eventos se transmitirán en tiempo real y, opcionalmente, se reenviarán a tu servidor local.
  </Step>

  <Step title="Envía un SMS">
      ```bash
      rewrite trigger sms.created -m "Hello from Rewrite" --to +5511999999999
      ```
  </Step>
</Steps>

## Principios de diseño
La CLI de Rewrite fue creada para ser:

- Explícita, sin comportamiento oculto
- Scriptable, con salida estructurada predecible
- Compatible con CI, totalmente no interactiva mediante flags
- Rápida, con tiempo de arranque mínimo y bajo consumo
