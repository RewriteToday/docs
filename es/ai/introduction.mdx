## Ciclo tipico de workflow AI

<Accordion title="1. Definir alcance">
  Define objetivo, limites de responsabilidad y criterios de aceptacion antes de ejecutar.
</Accordion>

<Accordion title="2. Ejecutar con una skill">
  Usa una skill enfocada para guiar discovery, implementacion y validacion con menos variacion.
</Accordion>

<Accordion title="3. Verificar resultados">
  Confirma comportamiento con tests, chequeos de docs y protecciones contra regresion.
</Accordion>

<Accordion title="4. Iterar la skill">
  Actualiza la skill con aprendizajes para que futuras ejecuciones sean mas rapidas y seguras.
</Accordion>
