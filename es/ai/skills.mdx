Las Skills de Rewrite son paquetes reutilizables de capacidades para agentes de AI.
Definen flujos enfocados, instrucciones reutilizables y patrones de ejecucion para entregar mas rapido con calidad predecible.

## Como instalar y usar Skills

Usa el siguiente comando para agregar una fuente de skills:

```bash
npx skills add <link>
```

Ejemplo con Skills de Rewrite:

```bash
npx skills add https://github.com/RewriteToday/skills
```

<Info>
Despues de agregar, sigue la documentacion de tu entorno de agente para listar, activar y ejecutar las skills instaladas.
</Info>

## Estructura tipica de una skill

<Accordion title="1. Alcance e intencion">
  Define exactamente que cubre la skill y cuando debe activarse.
</Accordion>

<Accordion title="2. Flujo de ejecucion">
  Enumera la secuencia: discovery, implementacion, validacion y handoff.
</Accordion>

<Accordion title="3. Guardrails">
  Agrega reglas obligatorias: seguridad, estilo y operaciones prohibidas.
</Accordion>

<Accordion title="4. Activos reutilizables">
  Incluye scripts, plantillas y referencias para reducir trabajo manual.
</Accordion>

## Buenas practicas

- Mantener cada skill enfocada en un solo resultado.
- Preferir pasos concretos sobre instrucciones genericas.
- Definir criterios de validacion medibles.
- Versionar skills junto con el codigo y la herramienta.
- Conectar cada skill con flujos reales del equipo.
