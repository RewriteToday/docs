## Eventos SMS

<div style={{ display: 'grid', gap: '26px' }}>

  <div>
    <div>— <a href="/es/webhooks/events/sms-queued"><code style={{ color: '#B5B3AD' }}>sms.queued</code></a></div>
    <div style={{ marginLeft: '18px', marginTop: '6px', opacity: 0.85 }}>
      Ocurre cuando un **SMS** es aceptado y colocado en la cola de entrega.
    </div>
  </div>

  <div>
    <div>— <a href="/es/webhooks/events/sms-delivered"><code style={{ color: '#3DD68C' }}>sms.delivered</code></a></div>
    <div style={{ marginLeft: '18px', marginTop: '6px', opacity: 0.85 }}>
      Ocurre cuando la operadora confirma la entrega correcta al dispositivo del destinatario.
    </div>
  </div>

  <div>
    <div>— <a href="/es/webhooks/events/sms-scheduled"><code style={{ color: '#FFCA16' }}>sms.scheduled</code></a></div>
    <div style={{ marginLeft: '18px', marginTop: '6px', opacity: 0.85 }}>
      Ocurre cuando un **SMS** se programa para enviarse en un momento futuro.
    </div>
  </div>

  <div>
    <div>— <a href="/es/webhooks/events/sms-failed"><code style={{ color: '#FF9592' }}>sms.failed</code></a></div>
    <div style={{ marginLeft: '18px', marginTop: '6px', opacity: 0.85 }}>
      Ocurre cuando el **SMS** falla de forma permanente por números inválidos, rutas bloqueadas, límites de cuota o rechazo del proveedor.
    </div>
  </div>

  <div>
    <div>— <a href="/es/webhooks/events/sms-canceled"><code style={{ color: '#BAA7FF' }}>sms.canceled</code></a></div>
    <div style={{ marginLeft: '18px', marginTop: '6px', opacity: 0.85 }}>
      Ocurre cuando un **SMS** programado se cancela antes de enviarse.
    </div>
  </div>

</div>
