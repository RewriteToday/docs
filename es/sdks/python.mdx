## Antes de empezar

- Necesitas tener Python instalado en tu máquina.
- Necesitas una API key de Rewrite con los permisos correspondientes.

<Steps>
    <Step title="Instala el SDK" titleSize="h3">
        Para instalar el SDK de Python, ejecuta el siguiente comando:
        
        ```bash
        pip install rewrite
        ```
    </Step>
    
    <Step title="Configura la variable de entorno REWRITE_API_KEY" titleSize="h3">
    Configura tu API key de Rewrite en las variables de entorno.
    
    ```text .env    
    REWRITE_API_KEY=rw...
    ```
    </Step>
    
    <Step title="Envía tu SMS" titleSize="h3">
    Ahora solo necesitas pasar el contenido del mensaje y listo.
    
    ```py main.py
    import os
    import rewrite
    
    rewrite.api_key = os.environ["REWRITE_API_KEY"]
    
    def send_sms(to: str):
        params: rewrite.Messages.SendOptions = {
            "to": to,
            "message": "Your OTP code is 478201",
        }
    
        return rewrite.messages.send(params)
    
    
    if __name__ == "__main__":
        message = send_sms("+1234567890")
        print(message)
    ```
    
    Puedes hacer de todo con nuestro SDK, sigue explorando.
    </Step>
</Steps>
