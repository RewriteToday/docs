<Card title="Skills" icon="sparkles" href="./skills" color="#C9CBD1">

## Cycle type d'un workflow AI

<Accordion title="1. Definir le scope">
  Definissez objectif, limites de responsabilite et criteres d'acceptation avant execution.
</Accordion>

<Accordion title="2. Executer avec une skill">
  Utilisez une skill ciblee pour guider discovery, implementation et validation avec moins de variance.
</Accordion>

<Accordion title="3. Verifier les resultats">
  Confirmez le comportement avec tests, checks docs et protections contre les regressions.
</Accordion>

<Accordion title="4. Iterer la skill">
  Mettez a jour la skill avec les apprentissages pour accelerer et securiser les executions suivantes.
</Accordion>
