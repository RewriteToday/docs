Les Skills Rewrite sont des packs reutilisables de capacites pour agents AI.
Elles definissent des workflows cibles, des instructions reutilisables et des modeles d'execution pour livrer plus vite avec une qualite previsible.

## Installer et utiliser les Skills

Utilisez la commande suivante pour ajouter une source de skills :

```bash
npx skills add <link>
```

Exemple avec les Skills Rewrite :

```bash
npx skills add https://github.com/RewriteToday/skills
```

<Info>
Apres ajout, suivez la documentation de votre environnement agent pour lister, activer et executer les skills installees.
</Info>

## Structure type d'une skill

<Accordion title="1. Scope et intention">
  Definissez precisement ce que la skill couvre et quand elle doit etre declenchee.
</Accordion>

<Accordion title="2. Workflow d'execution">
  Decrivez la sequence: discovery, implementation, validation et handoff.
</Accordion>

<Accordion title="3. Guardrails">
  Ajoutez les regles non negociables: securite, style et operations interdites.
</Accordion>

<Accordion title="4. Assets reutilisables">
  Incluez scripts, templates et references pour limiter le travail repetitif.
</Accordion>

## Bonnes pratiques

- Garder chaque skill focalisee sur un seul resultat.
- Preferer des etapes concretes aux instructions generiques.
- Definir des criteres de validation mesurables.
- Versionner les skills avec le code et les outils.
- Relier chaque skill a des workflows reels de l'equipe.
