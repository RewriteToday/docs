## Avant de commencer

- Vous devez avoir Swift 5.9+ installé sur votre machine.
- Vous devez disposer d’une Rewrite API key avec les permissions requises.

<Steps>
    <Step title="Installer le SDK" titleSize="h3">
        Ajoutez la dépendance de package Swift :

        ```swift Package.swift
        dependencies: [
          .package(url: "https://github.com/rewritetoday/swift-sdk", from: "1.0.0")
        ]
        ```
    </Step>

    <Step title="Configurer la variable d’environnement REWRITE_API_KEY" titleSize="h3">
    Configurez votre Rewrite API key dans vos variables d’environnement.

    ```text .env
    REWRITE_API_KEY=rw...
    ```
    </Step>

    <Step title="Envoyer votre SMS" titleSize="h3">
    Maintenant, il suffit de passer le contenu du message et c’est tout.

    ```swift main.swift
    import Foundation
    import RewriteSDK

    @main
    struct Main {
      static func main() async throws {
        let rewrite = Rewrite(apiKey: ProcessInfo.processInfo.environment["REWRITE_API_KEY"] ?? "")

        let message = try await rewrite.messages.send(
          to: "+1234567890",
          message: "Your OTP code is 478201"
        )

        print(message)
      }
    }
    ```

    Vous pouvez tout faire avec notre SDK, continuez à explorer.
    </Step>
</Steps>
