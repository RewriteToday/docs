## Avant de commencer

- Vous devez avoir Kotlin/JVM installé sur votre machine.
- Vous devez disposer d’une Rewrite API key avec les permissions requises.

<Steps>
    <Step title="Installer le SDK" titleSize="h3">
        Ajoutez la dépendance du SDK Kotlin :

        ```kotlin build.gradle.kts
        dependencies {
          implementation("io.github.rewritetoday:rewrite-kotlin:x.y.z")
        }
        ```
    </Step>

    <Step title="Configurer la variable d’environnement REWRITE_API_KEY" titleSize="h3">
    Configurez votre Rewrite API key dans vos variables d’environnement.

    ```text .env
    REWRITE_API_KEY=rw...
    ```
    </Step>

    <Step title="Envoyer votre SMS" titleSize="h3">
    Maintenant, il suffit de passer le contenu du message et c’est tout.

    ```kt main.kt
    import com.rewritetoday.sdk.Rewrite
    import com.rewritetoday.sdk.models.messages.SendOptions

    fun main() {
      val rewrite = Rewrite(System.getenv("REWRITE_API_KEY"))

      val params = SendOptions(
        to = "+1234567890",
        message = "Your OTP code is 478201"
      )

      val message = rewrite.messages.send(params)
      println(message)
    }
    ```

    Vous pouvez tout faire avec notre SDK, continuez à explorer.
    </Step>
</Steps>
