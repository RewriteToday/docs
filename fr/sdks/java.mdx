## Avant de commencer

- Vous devez avoir Java 17+ installé sur votre machine.
- Vous devez disposer d’une Rewrite API key avec les permissions requises.

<Steps>
    <Step title="Installer le SDK" titleSize="h3">
        Ajoutez le SDK Java à votre projet :

        <CodeGroup>

        ```xml Maven
        <dependency>
          <groupId>io.github.rewritetoday</groupId>
          <artifactId>rewrite-java</artifactId>
          <version>x.y.z</version>
        </dependency>
        ```

        ```gradle Gradle
        implementation("io.github.rewritetoday:rewrite-java:x.y.z")
        ```

        </CodeGroup>
    </Step>

    <Step title="Configurer la variable d’environnement REWRITE_API_KEY" titleSize="h3">
    Configurez votre Rewrite API key dans vos variables d’environnement.

    ```text .env
    REWRITE_API_KEY=rw...
    ```
    </Step>

    <Step title="Envoyer votre SMS" titleSize="h3">
    Maintenant, il suffit de passer le contenu du message et c’est tout.

    ```java Main.java
    import com.rewritetoday.sdk.Rewrite;
    import com.rewritetoday.sdk.models.messages.SendOptions;

    public class Main {
      public static void main(String[] args) throws Exception {
        var rewrite = new Rewrite(System.getenv("REWRITE_API_KEY"));

        var params = SendOptions.builder()
          .to("+1234567890")
          .message("Your OTP code is 478201")
          .build();

        var message = rewrite.messages().send(params);
        System.out.println(message);
      }
    }
    ```

    Vous pouvez tout faire avec notre SDK, continuez à explorer.
    </Step>
</Steps>
