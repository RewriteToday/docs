La **Rewrite CLI** est l’outil officiel en ligne de commande pour interagir avec Rewrite directement depuis votre terminal.

Conçue pour les développeurs, elle vous permet d’envoyer des SMS, de gérer des flux de renvoi, d’écouter des webhooks, de simuler des livraisons et d’automatiser des pipelines de messagerie, le tout sans quitter votre shell.

## Installation

<CodeGroup>
```bash Go (recommandé)
go install github.com/rewritetoday/cli@latest
```
```bash Brew
brew install rewrite-cli
```
```bash Docker
docker pull rewrite/cli
```
</CodeGroup>

## Vérifier l’installation

```bash
rewrite --version
```

<Steps>
  <Step title="S’authentifier">
      ```bash
      rewrite login
      ```
      Authentifiez-vous à l’aide de votre **API key** ou du **flux OAuth**.
  </Step>
  <Step title="Écouter les webhooks">
    ```bash
    rewrite listen --forward-to http://localhost:3000/webhooks/rewrite
    ```
    Les événements seront diffusés en temps réel et peuvent être transférés vers votre serveur local.
  </Step>

  <Step title="Envoyer un SMS">
      ```bash
      rewrite trigger sms.created -m "Hello from Rewrite" --to +5511999999999
      ```
  </Step>
</Steps>

## Principes de conception
La Rewrite CLI a été conçue pour être :

- Explicite — Aucun comportement caché
- Scriptable — Sortie structurée prévisible
- Compatible CI — Entièrement non interactive via des flags
- Rapide — Temps de démarrage minimal et faible empreinte
